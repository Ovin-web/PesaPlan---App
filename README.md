# PesaPlan – Smart Budget Manager App

## Overview

PesaPlan is a Flutter-based personal finance and budgeting application designed to help users track income, expenses, savings goals, and overall financial behavior in one structured system.

The application combines budgeting tools with AI-driven insights and analytics to support better financial decision-making.

---

## Features

* Income and expense tracking
* Budget creation and monitoring
* Savings goals management
* Monthly financial comparison
* AI-powered financial assistant
* Voice input for quick transactions
* User authentication system
* Firebase integration
* Offline local storage support

---

## AI Capabilities

* Spending pattern analysis
* Budget recommendations
* Expense insights
* Predictive financial suggestions

---

## Tech Stack

* Flutter (Dart)
* Firebase Authentication
* Cloud Firestore
* SQLite (local database)
* Provider (state management)
* AI service integration

## Project Structure


## Getting Started

### Clone the repository

```bash
git clone https://github.com/Ovin-web/PesaPlan---App.git
```

### Navigate into project directory

```bash
cd PesaPlan---App
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
